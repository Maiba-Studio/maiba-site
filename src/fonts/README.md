# Fonts

Place `Recoleta-Bold.woff2` in this directory.

Recoleta is a premium typeface by Latinotype. You can purchase it from:
- https://www.latinotype.com/display-fonts/recoleta
- https://fonts.adobe.com/fonts/recoleta

Until the font file is added, the site falls back to DM Serif Display from Google Fonts.
